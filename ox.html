    • Что такое React

    React - это JavaScript библиотека для создания пользовательских интерфейсов.

    При помощи React можно создавать приложения как для браузеров, так и для мобильных устройств.

    Lesson 1
    Одностраничные приложения - SPA (single page application).

    L1. SPA

    В одностраничном приложении клиент запрашивает только одну страницу HTML и JS, CSS для всего приложения.

    При навигации по "страницам" весь визуал меняет сам клиент при помощи JavaScript.

    Приемущества SPA

    • Странца не перезагружается при навигации по приложению.
    • Моментальная перерисовка элементов страницы.
    • Легче разворачивать, т.к. SPA содержит только несколько статических файлов.
    • Эффективное использование пропускной способности.

    Недостатки SPA

    • Для поисковой оптимизации необходимо выполнить дополнительную настройку сервера для корректного сканирования ботами, такими ка Googlebot.
    • Большой размер файла JavaScript влияет на время начальной загрузки приложения.

    Lesson 3
    JSX.

    L3. JSX

    JSX — это специальный XML-подобный синтаксис технологии React, который позволяет очень просто описывать пользовательский интерфейс.

    Несмотря на то, что JSX очень похож на HTML, он им не является. Это расширение синтаксиса JavaScript, которое во время компиляции транслируется в обычный JavaScript.

    В процессе выполнения теги в JSX преобразуются в React-элементы. React-элементы — это то, из чего сделаны React-компоненты. По сути, React-элементы — это простые JavaScript-объекты.

    React использует эти объекты для создания DOM-модели и поддержания её в актуальном состоянии.
