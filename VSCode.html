Настройка VSCode

В этом разделе представлены настройки VSCode для удобства и упрощения работы в этом редактооре.
  • Russian language pack
    Переводит интерфейс на русский язык.
  • Live server
    Это расширение отрывает html файл прямо в браузере, что позволяет видеть изменения кода прямо онлайн.
  • Indent rainbow
    Подсвечивает отступы разными цветами для лучшей читаемости кода.
  • Image preview
    Показывает мини изображения напротив тегов или стилей.
  • htmltagwrap
    Позволяет обернуть выделенный код html в любой тег.
  • CSS tractor
    Позволяет из выделенного кода html вычленить все классы и вставить их в css файл.
  • CSS peek
    Позволяет посмотреть примененные стили наведясь на тег html.
  • CSS formater
    Позволяет форматировать CSS файл.
  • Code runner
    Позволяет запускать код прямо в редакторе во встроенном терминале..
